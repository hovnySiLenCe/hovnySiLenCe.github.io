题面

给出$n*m$个矩阵，从第一列开始，一列一列访问元素，依次记元素为$S_1,S_2,…S_{nm}$，则$k$的价值为相邻两个元素差的绝对值的最小值，即$k=min{|S_1 - S_2|,|S_2 - S_3|,…,|S_{nm-1} - S_{nm}|}$

现在可以改变任意行的顺序，求$k$的最大值

思路：

正解应该是状压DP

但是以下讲的是另一种玄学做法——搜索

然后做一些预处理

比如：

1、改变的是行的顺序，因此任意两行之间的最小差值可以预先求出来
